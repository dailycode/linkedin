# Soon generator
An Boilerplate for start my projects

## Getting Starter

**Requires:**
- [Node](https://nodejs.org/en/)
- [Gulp](http://gulpjs.com/)

#### Using
```sh
# Clone repository
$ git@github.com:andersonweb/soon-generator.git name_project
$ cd name_project

# Install dependencies
$ npm install

# Run
$ gulp
```
